# Carrot-Slicer_Profiles
(Orca)Slicer profiles for Denhac's big printer


To install:
 1. copy the "OrcaSlicer" folder into your user profile folder.

    Windows:   %appdata%     (C:\<username>\appdata\roaming\)
    Mac:   /Users/<username>/Library/        (Press Cmd-shift-. to show hidden files)

 2. Restart Orcaslicer
 3. Start the Add Printer wizard by clicking the button to the right of "Printer" in Orca
 4. Scroll down to the section labeled "Denhac" and select Carrot, then Next
 5. On the filament page, select all the "Generic" profiles desired, then Finish.
        This will import the custom filament rules.  Always use the Generic "@Carrot" filaments as a base.  
 6.  Happy Printing!
